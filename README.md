### Step 1: Create a GitHub Repository

1. Go to [GitHub.com](https://github.com) and log in
2. Click the "+" icon in the top right and select "New repository"
3. Name it something like `turkey-hunt-game`
4. Make it Public
5. Click "Create repository"

### Step 2: Upload Your File

Option A - Using GitHub Website:
1. Click "uploading an existing file"
2. Drag and drop the `index.html` file
3. Click "Commit changes"

Option B - Using Git Command Line:
```bash
git clone https://github.com/YOUR-USERNAME/turkey-hunt-game.git
cd turkey-hunt-game
# Copy index.html into this folder
git add index.html
git commit -m "Add Turkey Hunt game"
git push
```

### Step 3: Enable GitHub Pages

1. In your repository, click "Settings"
2. Scroll down and click "Pages" in the left sidebar
3. Under "Source", select "main" branch
4. Click "Save"
5. Wait a minute or two for deployment

### Step 4: Share Your Link

Your game will be live at:
```
https://YOUR-USERNAME.github.io/turkey-hunt-game/
```

Share this link with your friends!
